# Инструмент OJS 3.3 Metadata Auto-Filler

Этот инструмент помогает автоматически заполнять метаданные (Заголовки, Аннотации, Авторов) в системе OJS 3.3, используя подготовленный JSON.

## Как использовать

1. **Подготовка:**
   - Откройте этот файл `ojs_auto_fill.js` в текстовом редакторе.
   - Скопируйте **весь** код.

2. **Запуск в браузере:**
   - Зайдите в панель администрирования OJS, на страницу редактирования нужной статьи.
     - Обычно это вкладка **"Публикация" (Publication)**.
   - Нажмите `F12` или `Ctrl+Shift+I` чтобы открыть инструменты разработчика.
   - Перейдите во вкладку **Console**.
   - Вставьте скопированный код и нажмите `Enter`.

3. **Работа с панелью:**
   - Слева появится панель **"OJS Auto-Filler"**.
   - Вставьте ваш JSON с данными в текстовое поле.
   - Нажмите **"1. Распознать JSON"**.

4. **Заполнение данных:**
   - **Метаданные (Title/Abstract):**
     - Убедитесь, что вы видите поля заголовка и аннотации (вкладка "Title & Abstract" в OJS).
     - Нажмите кнопку **"Заполнить Заголовок и Аннотацию"**.
     - Скрипт попытается найти поля для RU, ENG и KAZ языков и заполнить их.
   - **Ключевые слова:**
     - В панели появятся поля с ключевыми словами для каждого языка.
     - Используйте кнопку **Copy**, чтобы быстро скопировать их и вставить в поле ввода OJS.
   - **Авторы (Contributors):**
     - Перейдите в раздел "Авторы" (Contributors) в OJS.
     - Для каждого автора:
       1. Нажмите "Изменить" (стрелочка рядом с именем -> Править/Edit) в OJS. Откроется модальное окно.
       2. В панели скрипта найдите этого автора и нажмите **"Заполнить этого автора"**.
       3. Скрипт заполнит Имя, Фамилию, Email, ORCID, Страну и Аффилиацию (Университет) на всех языках (переключаясь по полям).
       4. Сохраните изменения в OJS и переходите к следующему.

## Технические детали
- Скрипт использует селекторы атрибутов `name="title[ru_RU]"` и т.д.
- Для аннотаций (TinyMCE) используется API `tinyMCE.editors`.
- Для стран используется маппинг. Если страна не выбирается, проверьте написание в `COUNTRY_MAP` в начале скрипта.
